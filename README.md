# _Portfolio Webpage_

#### By _**Patrick Dolan**_

#### _A portfolio website including my projects and background information._

## Technologies Used

* _HTML_
* _CSS_
* _Bootstrap 4.5.3_
* _JQuery 3.6.0_

## Description

_A portfolio website for myself. This site includes information on my: coding projects, education, job experience, and hobbies. I heavily used cards to compartmentalize everything and even threw in a little Javascript with a list group to show my hobbies off._

## Setup/Installation Requirements

* _Clone the repository into a directory on your computer._
* _Navigate into the directory named porfolio and double click index.html to launch the website._
